**Q: 何か環境が必要？**
A: [BEGINNER_GUIDE.md](BEGINNER_GUIDE.md#準備するもの) で確認できます。基本的には PowerShell または ターミナルがあれば OK です

**Q: どのファイルから読み始めればいい？**
A: [00-START-HERE.md](00-START-HERE.md) から始めてください

**Q: 急いでいる場合は？**
A: [QUICKSTART.md](QUICKSTART.md) で最小限の実行が可能です

**Q: スクリプトをカスタマイズしたい**
A: [IMPLEMENTATION_GUIDE.md](IMPLEMENTATION_GUIDE.md) を参照

**Q: GitHub で公開したい**
A: [GIT_CONFIG_GUIDE.md](GIT_CONFIG_GUIDE.md) と [CONTRIBUTING.md](CONTRIBUTING.md) を確認
